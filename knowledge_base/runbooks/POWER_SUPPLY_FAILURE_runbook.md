# Runbook: Power Supply Failure

## Incident Type
Power Supply Failure

## Description
Failure in power delivery causing node shutdown.

## Symptoms
- power fluctuation
- sudden shutdown

## Impact
- node outage
- data loss risk

## Severity Levels
- Single failure: High
- Repeated: Critical

## Diagnostic Steps
1. Verify power logs.
2. Identify affected nodes.

## Remediation Options
1. Isolate node.
2. Replace power supply.

## Preferred Action Logic
Immediate isolation required.

## Escalation Policy
Immediate escalation to infrastructure team.
